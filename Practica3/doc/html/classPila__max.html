template<class T>
class Pila_max< T >

T.D.A Pila_max.

La clase abstracta Pila_max está formada por una lista de pares de elementos <T,T> con funcionamiento LIFO (Last In, First Out). En una pila, las operaciones de insercción y borrado de elementos tiene lugar en uno de los extremos denominado tope.

Si datos.size() == 0, la pila estará vacia.

El espacio requerido para el almacenamiento es 0(n), donde n es el numero de elementos de la Pila_max.
